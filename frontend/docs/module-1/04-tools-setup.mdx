# Chapter 4: Development Tools & Environment Setup

## Introduction

Professional robotics development requires a properly configured environment with the right tools. This chapter guides you through setting up your development workspace for robot programming.

## Installing Python

### Download Python

1. Visit [python.org](https://www.python.org/downloads/)
2. Download Python 3.12 or later
3. **Important**: Check "Add Python to PATH" during installation

### Verify Installation

```bash
# Check Python version
python --version
# Output: Python 3.12.0

# Check pip (package manager)
pip --version
# Output: pip 23.3.1 from ...

# Check Python location
python -c "import sys; print(sys.executable)"
```

## Virtual Environments

### Why Virtual Environments?

Virtual environments isolate project dependencies:
- Different projects can use different package versions
- Avoids conflicts between global packages
- Easy to reproduce development setup

### Creating a Virtual Environment

```bash
# Navigate to your project directory
cd my_robotics_project

# Create virtual environment (Linux/Mac)
python3 -m venv venv

# Create virtual environment (Windows)
python -m venv venv
```

### Activating the Virtual Environment

**Linux/Mac:**
```bash
source venv/bin/activate
# Prompt changes to: (venv) user@machine:~$
```

**Windows Command Prompt:**
```bash
venv\Scripts\activate
# Prompt changes to: (venv) C:\path\to\project>
```

**Windows PowerShell:**
```powershell
venv\Scripts\Activate.ps1
```

### Installing Packages

Once activated, install packages locally:

```bash
# Install single package
pip install pybullet

# Install multiple packages
pip install numpy scipy matplotlib

# Install specific version
pip install tensorflow==2.13.0

# List installed packages
pip list

# Freeze requirements to file
pip freeze > requirements.txt
```

### Using Requirements Files

Create `requirements.txt` for your project:

```
numpy==1.24.3
scipy==1.11.2
matplotlib==3.7.2
pybullet==3.2.6
opencv-python==4.8.0
PyQt5==5.15.9
```

Install all dependencies:

```bash
pip install -r requirements.txt
```


```bash

git clone https://github.com/username/robot-repo.git
```

## IDE Setup

### PyCharm Community Edition

1. Download from [jetbrains.com](https://www.jetbrains.com/pycharm/)
2. Install and open
3. Configure Python interpreter:
   - File → Settings → Project → Python Interpreter
   - Select virtual environment

### Visual Studio Code

1. Install from [code.visualstudio.com](https://code.visualstudio.com/)
2. Install Python extension (Microsoft)
3. Create `.vscode/settings.json`:

```json
{
    "python.defaultInterpreterPath": "${workspaceFolder}/venv/bin/python",
    "python.linting.enabled": true,
    "python.linting.pylintEnabled": true,
    "python.formatting.provider": "black",
    "editor.formatOnSave": true
}
```

### Creating a Launch Configuration

Create `.vscode/launch.json` for debugging:

```json
{
    "version": "0.2.0",
    "configurations": [
        {
            "name": "Python: Current File",
            "type": "python",
            "request": "launch",
            "program": "${file}",
            "console": "integratedTerminal"
        },
        {
            "name": "Python: Simulation",
            "type": "python",
            "request": "launch",
            "program": "${workspaceFolder}/src/simulation.py",
            "console": "integratedTerminal"
        }
    ]
}
```

## Writing Your First Script

Create `src/simple_robot.py`:

```python
#!/usr/bin/env python3
"""
Simple robot controller example.
Demonstrates basic robot control structure.
"""

import numpy as np
import pybullet as p
import pybullet_data
import time


class SimpleRobot:
    """A simple robot controller."""

    def __init__(self, robot_urdf="r2d2.urdf"):
        """Initialize robot in simulation."""
        # Connect to physics engine
        self.client = p.connect(p.GUI)
        p.setGravity(0, 0, -9.81)
        p.setAdditionalSearchPath(pybullet_data.getDataPath())

        # Load objects
        self.plane = p.loadURDF("plane.urdf")
        self.robot_id = p.loadURDF(robot_urdf, [0, 0, 1])
        self.num_joints = p.getNumJoints(self.robot_id)

    def get_joint_angles(self):
        """Get all joint angles."""
        angles = []
        for i in range(self.num_joints):
            state = p.getJointState(self.robot_id, i)
            angles.append(state[0])
        return np.array(angles)

    def set_joint_targets(self, targets):
        """Set target angles for joints."""
        for i, target in enumerate(targets):
            if i < self.num_joints:
                p.setJointMotorControl2(
                    self.robot_id,
                    i,
                    p.POSITION_CONTROL,
                    targetPosition=target,
                    force=100
                )

    def step(self):
        """Step simulation."""
        p.stepSimulation()

    def close(self):
        """Close simulation."""
        p.disconnect()


if __name__ == "__main__":
    # Create robot
    robot = SimpleRobot()

    # Simulate
    for step in range(1000):
        # Get current angles
        angles = robot.get_joint_angles()

        # Set target (simple sine wave movement)
        targets = np.sin(step / 100) * 0.5
        if robot.num_joints > 0:
            robot.set_joint_targets([targets] * robot.num_joints)

        robot.step()
        time.sleep(1.0 / 240.0)

        if step % 100 == 0:
            print(f"Step {step}: Angles = {angles[:3]}")

    robot.close()
```

Run your script:

```bash
python src/simple_robot.py
```

## Testing Your Setup

Create `tests/test_setup.py`:

```python
"""Test that all dependencies are installed correctly."""

import sys


def test_imports():
    """Test that key packages can be imported."""
    try:
        import numpy
        print("✓ NumPy installed")
    except ImportError:
        print("✗ NumPy NOT installed")
        return False

    try:
        import scipy
        print("✓ SciPy installed")
    except ImportError:
        print("✗ SciPy NOT installed")

    try:
        import pybullet
        print("✓ PyBullet installed")
    except ImportError:
        print("✗ PyBullet NOT installed")
        return False

    try:
        import matplotlib
        print("✓ Matplotlib installed")
    except ImportError:
        print("✗ Matplotlib NOT installed")

    print(f"\nPython version: {sys.version}")
    return True


if __name__ == "__main__":
    success = test_imports()
    sys.exit(0 if success else 1)
```

Run tests:

```bash
python tests/test_setup.py
```

## Troubleshooting

### Common Issues

**PyBullet GUI not opening (Linux)**
```bash
# Install missing display libraries
sudo apt-get install libgl1-mesa-glx libxkbcommon-x11-0

# Run with DISPLAY set
DISPLAY=:0 python script.py
```

**Permission Denied on Linux**
```bash
# Make script executable
chmod +x script.py

# Run directly
./script.py
```

**Module Not Found**
```bash
# Make sure virtual environment is activated
source venv/bin/activate

# Reinstall package
pip install --force-reinstall package_name
```
